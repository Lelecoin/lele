# Let's Get Started
Click on the LELE icon, or start from the command line:
```sh
Unix
./start.sh
```
```sh
Mac
./run.command
```
```sh
Windows
run.bat
```
Wait for the JavaFX wallet window to open
On platforms without JavaFX, open http://localhost:27876/ in a browser
